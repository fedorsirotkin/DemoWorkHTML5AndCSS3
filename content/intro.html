Введение в методологии и инструменты веб–разработки

Движки веб–браузеров

Браузерный движок (англ. layout engine) – представляет собой программу, преобразующую содержимое веб–страниц (файлы HTML, XML, цифровые изображения и т. д.) и информацию о форматировании (в форматах CSS, XSL и т. д.) в интерактивное изображение форматированного содержимого на экране.

GIT

Git (произн. «гит») – распределённая система управления версиями.

Пакетные менеджеры

Система управления пакетами - набор программного обеспечения, позволяющего управлять процессом установки, удаления, настройки и обновления различных компонентов программного обеспечения.

Текстовые редакторы и интегрированные среды разработки

Текстовый редактор – самостоятельная компьютерная программа или компонент программного комплекса (например, редактор исходного кода интегрированной среды разработки или окно ввода в браузере), предназначенная для создания и изменения текстовых данных в общем и текстовых файлов в частности.

Интегрированная среда разработки, ИСP/IDE (англ. Integrated development environment) – комплекс программных средств, используемый программистами для разработки программного обеспечения (ПО).
